关于CKT区块链

联盟体系

CKT区块链车辆生态联盟包含多家车辆行业生产商、经销商、第三方服务商、保险。

CKT通证平台

将CKT区块链生态联盟中所有构成部分联系起来，所有链上数据可操作，所有服务可通过CKT通证支付结算。

车辆数据标识

从生产到销毁，车辆行为数据上链全记录，授权奖励型机制，帮助锚定车辆价值，提高联盟内效率与数据真实性。

数字身份标识

联盟成员信息数据上链，所有者授权，保证数据来源真实性，协同高效性。
